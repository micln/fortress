# 运行手册

## 本地运行

1. 使用 Godot 4.6.x 打开仓库根目录。
2. 等待项目导入完成。
3. 从主场景启动运行。
4. 在竖屏窗口尺寸下检查地图、城市、UI 和输入。

## 调试建议

- 先验证主场景是否可启动，再检查输入和战斗逻辑。
- 如果战斗结果异常，优先排查领域层规则，而不是先改 UI。
- 如果点击无效，先检查输入层是否正确识别触控和鼠标事件。
- 如果行军部队穿模或直接穿过彼此，优先检查路上遭遇战逻辑与行军碰撞判定。

## 验证清单

- 竖屏显示是否正常。
- 城市是否能随机生成并连接道路。
- 被占领城市是否按秒产兵。
- 玩家与多个电脑势力是否都能按规则发起进攻。
- 不同势力的行军单位在道路上相遇时是否会先战斗抵消。
- 战斗结算是否符合 `docs/gameplay.md`。
