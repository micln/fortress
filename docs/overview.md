# 原型概述

`Fortress War` 是一个竖屏城池攻防原型，目标是用尽量少的操作，把“占领城市、扩张士兵、推进战线、道路遭遇战”这条核心循环跑通。

## 设计目标

- 适配手机竖屏，单手触控优先。
- 鼠标点击与触控保持同一套交互逻辑。
- 地图随机生成，保证每局布局不同。
- 战场应有超出单屏的纵深，允许玩家拖拽浏览全图。
- 规则简单明确，方便快速验证战斗节奏。
- 城市经营操作尽量就地完成，避免频繁弹窗打断主战场观察。
- 多个电脑势力彼此攻伐，让战线变化不只围绕玩家与单一敌军。

## 当前体验边界

- 先做核心循环，不把资源管理、技能、兵种克制一次性塞进来。
- 城市连接关系先以道路网络为主，后续再考虑更复杂的寻路与多段进攻。
- 电脑 AI 先保证“能进攻、会扩张、会互相争夺”，再逐步增强策略。
- 当前只支持大地图平移浏览，不引入缩放和迷你地图。
