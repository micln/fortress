# 产品文档

## 产品定位

`Fortress War` 是一个手机竖屏的轻策略攻防原型。玩家通过最少的点按操作，在随机地图上占城、运兵、扩张，并与多个独立电脑势力竞争最后统治权。

## 目标用户

- 喜欢轻量即时策略、但不希望被复杂资源系统拖慢节奏的玩家
- 主要在手机上体验，希望单手触控即可完成核心操作的玩家
- 需要快速验证规则闭环与战斗反馈的原型测试用户

## 核心体验

- 进入对局后，快速理解“城市会产兵、道路决定进攻、路上部队会遭遇战”这条主循环
- 玩家通过两次点击和一个出兵面板，即可完成进攻或运兵
- 多个电脑势力彼此攻伐，地图战线会持续变化，不是单纯的玩家打一个固定敌人
- UI 维持竖屏、低认知负担、高可读性，暂停和说明不应打断主战场观察

## 当前功能清单

- 随机生成连通地图、城市和道路
- 城市使用古代中国城市名随机命名
- 玩家 + 多个独立电脑势力 + 中立城市的多方混战
- 城市等级和人口上限
- 占领城市自动产兵，中立城市不产兵
- 玩家可跨图运兵；进攻必须沿道路相邻
- 出兵面板支持手动调兵、快捷按钮和结果预判
- 行军单位具象化展示，行军时间与距离相关
- 路上不同势力部队相遇会先战斗抵消
- 到城后再结算攻城或增援
- 电脑势力支持总方数、难度、风格配置
- 背景音乐与操作音效
- 暂停、重新开始、开始说明、结束提示

## 交互原则

- 优先保证单指点击和鼠标左键一致
- 操作反馈要及时，尽量通过顶部状态栏与音效提示玩家发生了什么
- 暂停应尽量轻量，不遮挡战场观察
- 城市图标和信息布局要在小尺寸下仍能快速识别

## 非目标

- 当前不引入资源采集、科技树、技能系统、兵种克制
- 当前不做联网、排行榜、存档
- 当前不做复杂经济系统和多段路径规划
